v4 — Flat Structure

No more cards in cards. Eliminated every wrapper card. Provider rows use bottom-border separation. Model options sit directly in the pane. Generation params are bare label+input pairs. Capabilities are simple list rows.

One border max. The only bordered containers are: the tab bar (structural), the flyout (floating), and form inputs. Everything else uses typography and spacing.

Hierarchy via Typography

Section labels (uppercase, dim, small) replace card wrappers for grouping: "OAuth", "API Keys", "Custom".

Dividers are gone between sections — whitespace does the job.

Collapsibles in Advanced tab are borderless <details> — just a summary line that rotates an arrow.

Custom Providers

The "Custom" section shows user-added providers as regular rows with hover-reveal edit/delete icons. The "+ Add Provider" button opens an inline form (no modal, no card wrapper — just fields separated by a top border).

Form fields: Name, Type (OpenAI/Anthropic-compatible), Base URL, API Key, Model IDs.

Data Model

providers: Record<id, ProviderInstance>

activeModel: { providerId, modelId }

modelOverrides: Record<key, GenParams>

roles: { orchestrator?, vision?, team[] }

preferences: { autoScroll, confirmActions, ... }
